Glossary

Amazon S3

Amazon Unlimited FTP Server. Can serve static websites. Cheap.

“Amazon Web Services, the “Powered by AWS” logo, AWS S3, AWS EC2, AWS Lambda, AWS ApiGateway are trademarks of Amazon.com, Inc. or its affiliates in the United States and/or other countries."

EC2

Amazon Virtual Servers. Can be used for anything that server is for. Expensive (pay for running server)

“Amazon Web Services, the “Powered by AWS” logo, AWS S3, AWS EC2, AWS Lambda, AWS ApiGateway are trademarks of Amazon.com, Inc. or its affiliates in the United States and/or other countries."

Lambda

Serverless backend. Can run almost any function in many programming languages. Cheap (pay only if invoked)

“Amazon Web Services, the “Powered by AWS” logo, AWS S3, AWS EC2, AWS Lambda, AWS ApiGateway are trademarks of Amazon.com, Inc. or its affiliates in the United States and/or other countries."

Api Gateway

Connects your services with world. You can easily add new REST methods to run your Lambda function
